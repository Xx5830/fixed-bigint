# fixed-bigint

**Шаблонный класс для знаковых целых чисел с фиксированной разрядностью.**

Полностью располагается на стеке (не использует динамическую память), написан на C++17, не имеет зависимостей, кроме стандартной библиотеки. Идеален для встраиваемых систем, криптографии, высокопроизводительных вычислений.

## Особенности
- **Header-only** — просто скопируйте `include/` в проект или используйте CMake.
- **Произвольная битность** на этапе компиляции: `fixed_bigint<256>`, `fixed_bigint<2025>`.
- Полный набор арифметики: `+`, `-`, `*`, `/`, `%`, унарный `-`.
- Битовые операции: `~`, `|`, `&`, `^`, сдвиги `<<`, `>>`.
- Сравнения: `==`, `!=`, `<`, `>`, `<=`, `>=`.
- Вывод в поток (`std::cout`), поддерживаются десятичный, шестнадцатеричный, восьмеричный форматы (через флаги `std::hex`, `std::oct`).
- Потокобезопасен

## Быстрый пример

```cpp
#include <fbi/fixed_bigint.hpp>
#include <iostream>

int main() {
    fbi::fixed_bigint<2025> a("123456789012345678901234567890");
    fbi::fixed_bigint<2025> b(42);
    std::cout << "a * b = " << (a * b) << std::endl;
    return 0;
}
